# Agent Skills Directory

This directory contains reusable Claude Code agent skills for the Physical AI Textbook project.

## Available Skills

### 1. Chapter Writer (`chapter-writer.md`)
Generates comprehensive textbook chapter content with proper formatting.

### 2. Code Example Generator (`code-example-generator.md`)
Creates runnable code examples for robotics concepts.

### 3. Quiz Generator (`quiz-generator.md`)
Generates practice questions and exercises.

### 4. Content Processor (`content-processor.md`)
Processes and indexes book content for RAG.

## Usage

Reference these skills in Claude Code to maintain consistency across content generation.
