# Quiz Generator Skill

## Purpose
Creates practice questions to reinforce learning from textbook chapters.

## Question Types

### 1. Conceptual Questions
Test understanding of core concepts.

### 2. Calculation Questions
Apply mathematical formulas.

### 3. Code Analysis
Analyze or debug code snippets.

### 4. Design Questions
Open-ended design challenges.

## Template

```markdown
## Practice Questions

### Conceptual
1. Explain the difference between [A] and [B].
2. Why is [concept] important for [application]?

### Calculation
3. Given [parameters], calculate [result].
4. If a robot arm has [specs], what is the [measurement]?

### Code Analysis
5. What does the following code output?
```python
[code snippet]
```

6. Find and fix the bug in this code:
```python
[buggy code]
```

### Design Challenge
7. Design a system that [requirement]. Sketch the architecture and explain your choices.
```

## Guidelines
- Mix difficulty levels (easy, medium, hard)
- Include at least one question per major section
- Provide answer hints where appropriate
- Make questions specific to chapter content
